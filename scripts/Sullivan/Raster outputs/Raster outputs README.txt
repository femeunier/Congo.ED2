Raster outputs readme

This folder contains rasters used to create Figure 4 and Figure S15.

File names indicate:
(1) whether the raster shows absolute change in carbon stocks (Delta carbon) or change as a percentage of original stocks (Delta carbon_percentage)
(2) The climate scenario investigated (RCP26 in 2050 or RCP45 in 2050)
(3) The CO2 fertilisation scenario (none, Terrer = from Terrer et al, Kolby = CMIP5 models from Kolby-Smith et al, FACE = FACE experiments from Kolby-Smith et al
