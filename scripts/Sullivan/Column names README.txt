Column names (MultiCensusPlots.csv - most also in AllPlots.csv)

PlotViewID - ForestPlots database ID for plot view
PlotCode - ForestPlots database plot code
AGB - Aboveground biomass (Mg/ha)
AGWP - Aboveground woody production (Mg/ha/yr)
Mort - Biomass mortality (Mg/ha/yr)
Lat - Latitude (degrees, rounded)
Long - Longitude (degrees, rounded)
Area - Plot area (ha)
Altitude - Plot altitude (m) - NA if field measured altitude not available
FirstCens - Date of first census
LastCens - Date of last census
MonitorLength - Monitoring length (years)
Midpoint - Midpoint date of period plot monitored for
Continent - Code indicating which continent plot is on
VPD - Vapour pressure deficit
VPDmax - Maximum monthly vapour pressure deficit
Sread - Incoming solar radiation
Srad.cv - Coefficient of variation amongst months of solar radiation
MTWM - Mean daily temperature of the warmest month
Alt.WC - Altitude of 1km grid-cell (resolution of climate data)
bio1-bio19 - Bioclimatic variables
Min.temp - Mean daily minimum temperature (degree C)
Max.temp - Mean daily maximum temperature (degree C)
WarmMin - Mean daily minimum temperature in the warmest month (degree C)
CloudSD - Standard deviation between months in cloud frequency (%)
Cloud - Cloud frequency (% of Modis passes with cloud)
Warm.rate1 - Warming rate (from CRU data) using linear regression
Warm.rate2 - Warming rate (from CRU data) using robust regression
mcwd - Maximum monthly cumulatative water decifit (mm)
ndry1 - Number of months where precipitation < 100 mm
ndry2 - Number of months with cumulatative water deficit
wind - Mean monthly wind speed
wind.max - Maximum monthly wind speed
CEC - Cation exchange capacity
Sand - % sand
Clay - % clay
pH - pH*10
Rock depth - predicted depth to R horizon
Rhoriz.prob - Probability of R horizon
MEM1-MEM8 - Moran's eigenvector maps
AGBChv_NoHt - Aboveground biomass (Mg/ha) calculated using Chave et al. 2005's no height equation
AGWPChv_NoHt - Aboveground woody production (Mg/ha/yr) calculated using Chave et al. 2005's no height equation
